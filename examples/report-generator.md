# Report Generator

## Overview

This module generates reports from transaction data stored in Google Sheets.

The reports are organized by category and reporting period to simplify monthly review.

---

## Process

1. Read transaction data.
2. Filter records by category.
3. Calculate totals.
4. Create or update the report sheet.
