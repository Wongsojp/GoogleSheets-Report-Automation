# GoogleSheets-Report-Automation

> Automated business reporting, monthly closing, backup, and archive management using Google Apps Script and Google Sheets.

---

## Overview

GoogleSheets-Report-Automation is a business reporting system built with Google Apps Script to automate repetitive financial and operational reporting tasks.

The project transforms raw transaction data stored in Google Sheets into structured monthly reports while automatically creating backups and organizing archived reports.

It is designed to reduce manual work, improve reporting accuracy, and standardize monthly closing processes.

---

## Key Features

### Monthly Closing

- Automated month-end closing process
- Closing validation
- Prevent accidental duplicate closing
- Opening balance generation

### Report Generation

- Automatic monthly reports
- Category-based reports
- Summary reports
- Transaction recaps

### Backup Automation

- Automatic spreadsheet backup
- Google Drive integration
- Version preservation
- Data protection before closing

### Archive Management

- Automatic archive creation
- Monthly folder organization
- File management
- Historical report storage

### Data Processing

- Automatic transaction grouping
- Monthly aggregation
- Category filtering
- Data validation

### User Interface

- Custom Google Sheets menu
- One-click report generation
- User notifications
- Progress logging
